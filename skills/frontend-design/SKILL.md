# Frontend Design

Create distinctive, production-grade frontend interfaces with exceptional aesthetic attention.

## Design Thinking

Before coding:
- **Purpose**: What problem does this solve? Who uses it?
- **Tone**: Pick an extreme - brutally minimal, maximalist, retro-futuristic, organic, luxury, playful, editorial, brutalist, art deco, etc.
- **Differentiation**: What makes this UNFORGETTABLE?

## Guidelines

- **Typography**: Choose distinctive fonts. Avoid Arial, Inter. Pair display font with refined body font.
- **Color & Theme**: Commit to cohesive aesthetic. CSS variables. Dominant colors with sharp accents.
- **Motion**: CSS animations, micro-interactions. Staggered reveals, scroll-triggering, hover states.
- **Spatial Composition**: Unexpected layouts, asymmetry, overlap, grid-breaking elements, negative space.
- **Backgrounds**: Gradient meshes, noise textures, geometric patterns, layered transparencies, grain overlays.

**NEVER** use: Inter, Roboto, Arial, purple gradients on white, predictable layouts. Make context-specific, genuinely designed choices.
